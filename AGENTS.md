# Purpose of this project/directory

- This directory is to be used for coding that has been delegated by Hermes.
- Any tool that it directs to be built or maintained should be kept here. 
- Whatever makes keeping this organized is sufficient. I presume single-file tools can be kept in this root directory but any that require multiple files or additional documentation should be maintained in their own sub-directory. 

# Guidelines

- All work done here should be done in a robust matter by ensuring it works and has well-written automated tests and specs. 
- Write specs using Gherkin, and implement the steps in whatever manner is most effective and efficient.
- Before concluding any tasks:
-- ensure any tests that might have been affected still work
-- review all relevant tests and make sure they are clean and updated 
-- if the automated specs are not sufficient make sure to smoke test the project to ensure it works
